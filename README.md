# To Setup <br/>
Clone or download this repository

1. cd meanClient <br/>
2. npm install <br/>
3. cd meanBackend<br/>
4. npm install

## To Run

+ To Run Mean Stack Frontend (angular) <br/>
ng serve --open

+ To Run Mean Stack Backend  <br/>
mongod - to start MongoDB shell (In cmd--users folder) <br/>
node index - to start node server
